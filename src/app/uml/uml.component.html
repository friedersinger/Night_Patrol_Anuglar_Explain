The system encompasses numerous drawable objects. The World entity can accommodate multiple drawable objects, while each drawable object can only be associated with a single world. Hence, this represents a One-To-Many relationship.

Inheritance is the ability to create classes based on other classes. With inheritance, we can define a parent class (with certain properties and methods), and then children classes that will inherit from the parent class all the properties and methods that it has.

super()

The super keyword is used to access properties on an object literal or class's [[Prototype]], or invoke a superclass's constructor. The super. prop and super[expr] expressions are valid in any method definition in both classes and object literals.
