Night Patrol Game

Night Patrol is an HTML-based game designed to run in a web browser. It features an immersive gameplay experience where players control a character navigating through different levels, avoiding enemies, collecting coins, and defeating bosses.

Demo

You can play the game by visiting night-patrol. Give it a try and see how far you can go!

Game Components

HTML Structure

The HTML structure defines the layout and elements of the game, including the canvas, buttons, images, and audio elements.

JavaScript Classes

  • World:

    Represents the game world and contains instances of other classes and game objects. It handles game logic, collision detection, sound effects, and rendering.

  • Character:

    Represents the player character in the game. It has properties and methods related to the character's position, movement, energy, and interactions with other game objects.

  • Endboss:

    Represents the end boss character in the game. It has properties and methods related to the boss's position, energy, and interactions with other game objects.

  • DrawableObject:

    Base class for objects that can be drawn on the canvas. It provides basic properties and methods for positioning and rendering objects.

  • MovableObject:

    Extends the DrawableObject class and adds properties and methods for objects that can move, such as enemies, clouds, and background objects.

  • Gangster:

    Represents enemy characters in the game. It extends the MovableObject class and adds methods for enemy-specific behavior.

  • Cloud:

    Represents cloud objects in the game. It extends the MovableObject class and adds methods for cloud-specific behavior.

  • StatusBar:

    Represents a status bar in the game, displaying the character's energy level. It provides methods for updating and rendering the status bar.

  • Fullscreen:

    Handles fullscreen functionality in the game.

  • BackgroundObject:

    Represents background objects in the game, such as scenery and decorations. It extends the DrawableObject class.

  • Keyboard:

    Handles keyboard input and tracks key states.

  • Level:

    Represents a game level and contains information about the level's objects, enemies, and coins.

  • Coins:

    Represents coin objects in the game. It extends the DrawableObject class and provides methods for coin-specific behavior.

  • WaterBomb:

    Represents water bomb objects in the game. It extends the DrawableObject class and provides methods for water bomb-specific behavior.

  • SmallCoin:

    Represents small coin objects in the game. It extends the DrawableObject class and provides methods for small coin-specific behavior.

  • SmallWaterBomb:

    Represents small water bomb objects in the game. It extends the DrawableObject class and provides methods for small water bomb-specific behavior.

  • ThrowableObject:

    Represents objects that can be thrown by the character, such as water bombs. It extends the DrawableObject class and provides methods for throwable object-specific behavior.

Usage

To run the Night Patrol game, simply open the provided HTML file in a web browser that supports JavaScript. The game will load, and you can start playing by following the on-screen instructions and using the keyboard controls.
